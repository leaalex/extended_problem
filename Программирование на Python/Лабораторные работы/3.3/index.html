Цель работы: разработать код на Python для обработки табличных данных различного формата.

Задание

В соответствии с примером, представленном в методических указаниях и скринкасте, провести обработку исходных данных и преобразовать их в единый файл формата «.csv».

Шаги выполнения задания

  1. Скачать исходные данные ($source_filename)
  2. Загрузить данные в блокнот Google Colab
  3. Провести преобразования данных для их объединения в один файл, согласно представленной в указаниях к лабораторной работе методике и скринкасту
  4. Сохранить итоговый файл в формате «.csv»
  5. Загрузить итоговый файл в формате «.csv» на платформу
  6. Файл ответа загрузите, воспользовавшись специальным полем загрузки ответов на странице задания.
