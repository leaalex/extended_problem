Цель: установить IFC-просмотрщик BIMvision, определять значения атрибутов элементов модели в файлах IFC

Рекомендации по выполнению задания

1. Установка IFC-просмотрщика BIMvision

Скачайте по ссылке установочные файлы BIMvision 2.28.1 Setup.exe

Запустите установщик

Примите лицензионное соглашение и продолжите установку

Нажмите Finish для завершения установки

Программа запустится автоматически

2. Скачайте по ссылке файл Convenience store_Система вентиляции.ifc

3. В BIMvision выберите «Файл», «Открыть»

Выберите скачанный файл Convenience store_Система вентиляции.ifc и нажмите «Открыть»

Откроется цифровая информационная модель системы вентиляции
