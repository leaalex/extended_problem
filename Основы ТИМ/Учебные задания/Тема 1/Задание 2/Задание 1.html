Откройте проект House, который поставляется в комплекте Renga. Определите и впишите параметр ската кровли в соответствующем поле ответа.

Рекомендации по выполнению задания

1. Запустите Renga (двойное нажатие на ярлык).

2. Нажмите Открыть.

3. Выберите проект с названием House, который расположен в папке Samples, путь по умолчанию C:\Program Files\Renga Professional\Samples.

4. Выделите крышу в осях 1-4, справа появится панель Параметры. В поле параметра с именем Смещение по вертикали отобразится значение.

Впишите правильный вариант ответа

 Чтобы появилась панель Параметры, необходимо выделить крышу. Название параметра отображается при наведении на него курсора мыши.
