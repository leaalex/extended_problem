>>Для проверки наличия значения, соответствующего определенному ключу, в словаре без возникновения ошибки при отсутствии данного ключа нужно использовать метод…<< ( ) 
.get()
{{Метод возвращает значение, соответствующее указанному ключу, если ключа нет, то возвращает None, что исключает возможность появления ошибки}} ( ) 
.keys()
{{Данный метод возвращает список, содержащий все ключи словаря, но не возвращает значения}} ( ) 
.items()
{{Таким образом вместо одного значения будут возвращены все пары «ключ – значение»}} ( ) 
.popitem()
{{Данный метод вернет последнюю пару «ключ – значение» в словаре и удалит ее}}
