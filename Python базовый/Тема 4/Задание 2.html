Выберите правильный вариант ответа 

print('{:<20}'.format('text'))
При таком форматировании выравнивание текста произойдет по левому краю с ограничением поля вывода двадцатью символами 
print('{:^15}'.format('text'))
При таком форматировании выравнивание текста произойдет по центру с ограничением поля вывода пятнадцатью символами 
print('{4:%}'.format('text'))
Данный тип форматирования можно применять только к числовым значениям для их отображения в формате процентов; в данном случае возникнет ошибка, связанная с индексом переданных аргументов в функцию .format() 
print('{:>6}'.format('text'))
При таком форматировании выравнивание текста произойдет по правому краю с ограничением поля вывода шестью символами
