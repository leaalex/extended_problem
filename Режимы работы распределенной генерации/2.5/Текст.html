Пример 1. Определить номинальную \(N_{ice}\) (с учетом потерь в генераторе) и максимальную \(N_{ice\_max}\) мощность двигателя для электростанции со следующими показателями: \(N_{en} = 1000 \ кВт\) (номинальная мощность электростанции); \(\eta_g = 0.9\) (КПД генератора); \(N_s = 15 \ кВт\) (расход мощности на привод агрегатов собственных нужд); \(N_{e\_max} = 1100 \ кВт\) (максимальная мощность электростанции). Также определить удельный расход топлива электростанции \(g_e\) и двигателя \(g_{ice}\), приводящего в действие эту электростанцию, с параметрами указанными выше, если за час работы двигатель расходует \(200 \ кг\) топлива \((G_f = 200 \ кг/ч)\).

Решение

Номинальная мощность двигателя для электростанции мощностью \(1000 \ кВт\):

\( N_{ice} = \cfrac{N_{en} + N_s}{\eta} = \cfrac{1000 + 15}{0.9} = 1128 \ кВт\).

Максимальная мощность, которую должен развивать двигатель с \(10 \ \%\) запасом мощности на перегруз:

\( N_{ice\_max} = \cfrac{N_{en} + 0.1 \cdot N_{en} + N_s}{\eta} = \cfrac{1000 + 0.1 \cdot 1000 + 15}{0.9} = 1235 \ кВт\).

Удельный расход топлива двигателя для привода электростанции мощностью \(1000 \ кВт\):

\( g_{ice} = G_f / N_{ice} = 200/1128 = 0.177 \ кг/(кВт·ч)\).

Удельный расход топлива электростанции мощностью \(1000 \ кВт\):

\( g_e = G_f / N_{en} = 200/1000 = 0.200 \ кг/(кВт·ч)\).

Удельный расход топлива у электростанции больше, чему удельный расход топлива у двигателя, т.к. при одном и том же количестве израсходованного топлива двигатель развивал большую мощность, чем электростанция, на величину потерь в генераторе и на затраты мощности на собственные нужны электростанции.
