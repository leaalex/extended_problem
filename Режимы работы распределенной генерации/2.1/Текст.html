Пример 1. Дизельный двигатель со степенью сжатия \(\varepsilon = 15\) работает при частоте вращения коленчатого вала \(n = 2000 \ мин^{-1}\). Определить величину максимального давления рабочего цикла \(p_{max}\), если известно, что степень повышения давления \(\lambda = 2.0\). Давления рабочего тела в конце такта впуска \(p_{a} = 0.095 \ МПа\).

Решение

При выполнении приближенного расчета рабочего цикла дизельного двигателя реальный процесс сгорания топлива заменяется процессом изохорно-изобарного (смешанного) подвода теплоты к рабочему телу (рис. 1).

Сгорание начинается в момент нахождения поршня в ВМТ. Давление начала сгорания \(p_c\), температура – \(T_c\). Часть топлива сгорает при постоянном объеме рабочего тела \((V = const)\). При этом давление возрастает от \(p_c\) до \(p_z\), которое \((p_z)\) соответствует концу процесса изохорного сгорания. Оставшаяся часть топлива сгорает при \(p = const\). Таким образом, давление \(p_z\) является максимальным давлением цикла. При этом соотношение величин \(p_c\) и \(p_z\) характеризуется степенью повышения давления \(\lambda = \cfrac{p_z}{p_c}\).

Для решения задачи необходимо найти величину давления рабочего тела в конце такта сжатия:

\[ p_c = p_a \varepsilon^{n_1}.\]

Показатель политропы сжатия рассчитываем по формуле:

\[ n_1 = 1.41 - \cfrac{100}{n} = 1.41 - \cfrac{100}{2000} = 1.36. \]

Тогда давление в конце сжатия:

\[p_c = 0.095 \cdot 15^{1.36} = 3.78 \ МПа.\]

Максимальное давление рабочего цикла:

\[p_{max} = p_c \lambda = 3.78 \cdot 2 = 7.56 \ МПа.\]

Пример 2. Бензиновый двигатель со степенью сжатия \(\varepsilon = 9.9\) работает при частоте вращения коленчатого вала \(5600 \ мин^{-1}\). Давление в конце такта впуска \(p_{a} = 0.088 \ МПа\). Определить величину максимального давления цикла \(p_{max}\) и давление рабочего тела в конце такта расширения \(p_{b}\).

Решение

При расчете теоретического рабочего цикла бензинового двигателя действительный процесс сгорания представляется в виде изохорного (рис. 2).

Процесс сгорания начинается в ВМТ при давлении \(p_c\) протекает при постоянном объеме \((V = const)\) и заканчивается в ВМТ при давлении \(p_z\). Величина \(p_z\) рассчитывается с использованием показателя степени повышения давления \(\lambda\).

Для бензинового двигателя \(\lambda = p_z/p_c = 4.0 ... 4.5\). Полученные таким методом значения \(p_z\) превышают величины \(p_max\) бензиновых двигателей, замеряемые экспериментальным путем. Для согласования расчетов с практикой применяют так называемое скругление идеализированной индикаторной диаграммы, полагая, что \(p_max = 0.85 p_z\).

Согласно опытным данным величина среднего показателя политропы расширения для бензиновых двигателей \(n_2 = 1.18 ... 1.28\), а для дизельных двигателей \(n_2 = 1.18 ... 1.24\).

Показатель политропы сжатия:

\[ n_1 = 1.41 - \cfrac{100}{n} = 1.41 - \cfrac{100}{5600} = 1.392. \]

Давление в конце такта сжатия:

\[ p_c = p_a \varepsilon^{n_1} = 0.088 \cdot 9.9^{1.392} = 2.14 \ МПа.\]

Давление в конце сгорания:

\[ p_z = p_c \lambda = 2.14 \cdot 4.3 = 9.2 \ МПа.\]

Максимальное давление цикла:

\[ p_{max} = 0.85 p_z = 0.85 \cdot 9.2 = 7.85 \ МПа.\]

Принимая значение показателя политропы расширения \(n_2 = 1.27\), рассчитываем давление в конце такта расширения:

\[ p_b = p_z \left(\cfrac{\nu_c}{\nu_a}\right)^{n_2} =\cfrac{p_z}{\varepsilon^{n_2}} = \cfrac{9.2}{9.9^{1.27}} = 0.5 \ МПа.\]
