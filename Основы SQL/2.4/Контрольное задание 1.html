Необходимо разработать SQL запрос, который выводит имена клиентов, совершивших минимум один заказ. Разработчик составил следующий запрос:

SELECT name FROM customers WHERE id IN (SELECT id FROM orders)

Однако данный запрос выводил информацию как о тех клиентах, которые совершали заказы, так и о тех, которые заказы не совершали. При этом не все клиенты, которые совершали заказы, попадали в вывод результатов запроса. Исправьте ошибку в запросе, составленном разработчиком. Для этого необходимо:

  • изучить схему базы данных онлайн-школы;
  • просмотреть пример данных таблиц Customers и Orders;
  • прочитать описание данных, которые нужно извлечь;
  • изучить составленный разработчиком SQL запрос и найти в нем ошибку;
  • с помощью конструктора операторов SQL составить правильный запрос.
