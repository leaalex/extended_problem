Разработчику было необходимо создать оператор SQL, который выводит информацию о продуктах и их типах, отсортированную по типам продуктов. Разработчик подготовил следующий запрос:

SELECT t.type_name, p.name 
FROM product_types AS t
JOIN products AS p
ON p.type_id = t.id
ORDER BY t.type_name

Однако данный запрос не выводит информацию о типах продуктов, для которых пока нет ни одного продукта.

Исправьте ошибку разработчика и создайте SQL-запрос, который выводит информацию о продуктах онлайн-школы, отсортированную по их типам, включая типы продуктов, для которых продукты пока не созданы. Для этого необходимо:

  • изучить схему базы данных онлайн-школы;
  • просмотреть пример данных таблиц Products и Product_types;
