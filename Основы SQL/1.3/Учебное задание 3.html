id INT Идентификатор продукта
name VARCHAR(50) Название продукта
type VARCHAR(50) Тип продукта
price INT Цена продукта

Составьте SQL запрос, который находит название все бесплатных вебинаров онлайн-школы. Для этого необходимо:

  • прочитать описание столбцов и типов данных таблицы Products;
  • просмотреть данные, которые содержатся в таблице Products;
  • с помощью конструктора операторов SQL составить оператор, который извлекает нужные данные из таблицы Products.
