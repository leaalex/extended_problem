Цель:

Применять математические подходы для определения основных параметров функционирования энергообъектов на основе ВИЭ в различных климатических/метеорологических условиях.

Задание

  1. Для заданной зависимости мощности ветроэнергетической установки от скорости ветра, распределению Вейбулла по скоростям ветра и максимальной электрической нагрузке определить число ветроэнергетических установок заданной установленной мощности, необходимое для обеспечения питания присоединенной нагрузки.
  2. С учетом результатов, полученных в п.1 [КИУМ, количество ВЭУ], определить суммарную годовую выработку электрической энергии ветровой электростанцией при заданном графике числе часов использования установленной мощности ветроэнергетических установок.

Исходные данные

  • зависимость мощности ветроэнергетической установки от скорости ветра, \(P(v)\), о.е.
  • плотность вероятности распределения Вейбулла по скоростям ветра, \(p(v)\), о.е.

Указания к выполнению задания

  1. Изучить указания к выполнению лабораторной работы, посмотреть видеопрактикум.
  2. На основе таблицы из исходных данных определить КИУМ ветроэнергетической установки (определить интеграл произведения двух функций: плотности распределения по скоростям ветра и зависимости мощности ветрогенератора от скорости ветра). Рекомендация: воспользоваться методами численного интегрирования, например методом трапеций).
  3. Рассчитать мощность генерации ветровой установки с учетом определенного КИУМ.
  4. Определить необходимое оптимальное число ветроэнергетических установок для обеспечения питания подключенной нагрузки (при получении дробного числа производить округления в большую сторону).
  5. Рассчитать суммарную годовую выработку электроэнергии на электростанции.
  6. Найденные значения округлить до целых значений в кВт∙ч.
  7. Результаты расчетов вписать в соответствующие ячейки.
