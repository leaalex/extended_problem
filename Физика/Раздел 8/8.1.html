Два автомобиля одновременно выезжают из одного места по двум взаимно перпендикулярным направлениям. Один со скоростью \(V_1 = $a \ км/ч\), другой со скоростью \(V_2 = $b \ км/ч\). Чему расстояние \(S\) между автомобилями в момент, когда первый автомобиль прошел путь \(S_1 = $c \ км\)? Ответ выразите в километрах, округлив результат до целого числа.

Рекомендации:

Воспользуйтесь формулой, определяющую относительную скорость движения двух тел. Далее определите время движения первого автомобиля. Используя формулу для нахождения пути при равномерном прямолинейном движении определите расстояние \(S\) между автомобилями. Введите ответ в километрах, округлив результат до целого числа.

Внимание! В качестве десятичного разделителя используйте ТОЧКУ.

Выполнить:

\(S = \) км.

Подсказка:

Относительная скорость определяется выражением:

\[ \overrightarrow{V}_{отн} = \overrightarrow{V}_{1} - \overrightarrow{V}_{2} \]

При определении относительной скорости необходимо учесть, что два автомобиля двигаются по двум взаимно перпендикулярным направлениям.

Для определения времени движения используйте формулу:

\[ t = \frac{S_1}{V_1}.\]

Используя полученные выражения и формулу для нахождения пути при равномерном прямолинейном движении определите расстояние \(S\) между автомобилями.

\[ S = \overrightarrow{V}_{отн} \cdot t.\]

Введите ответ в километрах, округлив результат до целого числа.

Решение:

Относительная скорость определяется выражением:

\[ \overrightarrow{V}_{отн} = \overrightarrow{V}_{1} - \overrightarrow{V}_{2} \]

Так как два автомобиля двигаются по двум взаимно перпендикулярным направлениям, то векторная диаграмма для нахождения относительной скорости будет выглядеть следующим образом:

По теореме Пифагора:

\[ V_{отн} = \sqrt{V^{2}_{1} + V^{2}_{2}}. \]

Определим время движения первого автомобиля:

\[ t = \frac{S_1}{V_1}. \]

Используя формулу для нахождения пути при равномерном прямолинейном движении определим расстояние \(S\) между автомобилями:

\[ S = V_{отн} \cdot t.\]

Далее подставим в формулу пути выражение для относительной скорости и получим:

\( S = \frac{S_1}{V_1} \sqrt{V^{2}_{1} + V^{2}_{2}} \)

\( S = \frac{$c}{$a} \sqrt{$a^{2} + $b^{2}} = $S \) км.
