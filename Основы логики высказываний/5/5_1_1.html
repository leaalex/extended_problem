Определите, находятся или не находятся предложенные ниже формулы PL между собой в фундаментальных логических отношениях. Укажите в таблице правильные варианты ответа.

Указания к выполнению задания:

  1. Формулы PL могут находиться или не находиться между собой в трех фундаментальных логических отношениях: быть совместимыми по истинности; быть совместимыми по ложности; логически следовать друг из друга.
  2. Фундаментальные логические отношения независимы друг от друга, то есть сочетание логических отношений, в которых находятся две произвольные пропозициональные формулы, может быть любым.
  3. Если непосредственное определение фундаментальных логических отношений, в которых находятся сравниваемые формулы, вызывает затруднение, постройте (например, ручкой на отдельном листке бумаги) для каждой из формул таблицу истинности и по результату столбцов с полученными значениями этих формул проведите сравнение с определениями совместимых (несовместимых) по истинности формул, совместимых (несовместимых) по ложности формул и находящихся (ненаходящихся) в отношении логического следования формул.
  4. Сохраните для себя составленные вами таблицы и правильные ответы, они будут необходимы для выполнения Учебного задания 2.
