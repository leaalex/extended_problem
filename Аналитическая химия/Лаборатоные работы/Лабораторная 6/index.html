При ААС-определении железа и меди в производственном растворе применили метод добавок с построением графика.

Для приготовления серии растворов в три мерные колбы вместимостью \(50.0 \ см^3\) ввели аликвоты \(10 \ см^3\) анализируемого и стандартных растворов (\(T_{\rm{Cu}} = 0.001 \ г/см^3\), \(T_{\rm{Fe}} = 0.0001 \ г/см^3\)) меди и железа, разбавили до метки раствором \(0.3 \ моль/л\) \(\rm{HNO_3}\), получили атомный пар и измерили оптические плотности.

Определите содержание железа и меди в анализируемом растворе (г), используя данные анализа таблицы 1.

Проведите статистическую обработку результатов измерений. Ответы в виде десятичной дроби впишите в соответствующие ячейки таблицы 2 и таблицы 3.

Цель:

научиться определять содержание железа и меди в растворе графическим методом добавок.

Указания к выполнению задания:

  1. Посмотрите видео лабораторная работа №6 «Обнаружение микроколичеств элементов в технологическом растворе методом ААС-анализа».
  2. Внимательно прочитайте условия задания.
  3. Рассчитайте средние значения абсорбции для всех растворов и концентрации добавок (\(C_{доб \ 1}, \ C_{доб \ 2}\)) в анализируемой пробе, используя данные таблицы 1.
  4. Найденные значения впишите в соответствующие ячейки таблицы 2 с точностью до \(0.0001\).
  5. Постройте графики по методу добавок для каждого определяемого металла. Для построения графика можно использовать миллиметровую бумагу или любой софт (например, Microsoft Excel).
  6. Определите по графику концентрации ионов металлов в пробе.
  7. Рассчитайте массы определяемых веществ в анализируемом растворе с учетом разбавления.
  8. Найденные значения масс в виде числа впишите в соответствующие ячейки таблицы 3 с точностью до \(0.01 \cdot 10^{-5} \ г\).
