Цель: строить UML-диаграммы по описанию

Задача. Составить код для построения UML-диаграммы по приведенному описанию.

УСД (устройство сбора данных) имеет три возможные модификации: УСД_А, УСД_М и УСД_У. УСД реализует интерфейс передачи пакетов данных ППД и использует систему передачи пакетов СПП. УСД может включать в себя датчики метеорологические МД и гидрологические ГД, которые относятся к классу Д.
