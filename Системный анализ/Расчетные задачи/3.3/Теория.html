Пример 1. Дано описание пользователей системы и ее функционала:

Editor (редактор)

  • Open (открыть файл)
  • View (просмотреть файл)
  • Edit (редактировать файл)
  • Save (сохранить файл)
  • Export (экспортировать файл)
  • Import (импортировать файл

Reader (читатель)

  • Open
  • Import
  • View

Необходимо построить UML-диаграмму использования автоматизировано, с помощью программного кода.

Решение

Представленный код можно вставить и запустить в среде Google Colab. В первую ячейку нужно поместить код для подключения библиотеки Graphviz:

import graphviz

Во вторую код, реализующий UML-диаграмму использования:

g = graphviz.Graph('UseCase')

g.node('Editor')
g.node('Reader')

with g.subgraph(name='cluster_0') as a:
  a.node('Open')
  a.node('View')
  a.node('Edit')
  a.node('Save')
  a.node('Export')
  a.node('Import')

g.edge('Editor', 'Open')
g.edge('Editor', 'View')
g.edge('Editor', 'Edit')
g.edge('Editor', 'Save')
g.edge('Editor', 'Export')
g.edge('Editor', 'Import')

g.edge('Reader', 'Open')
g.edge('Reader', 'View')
g.edge('Reader', 'Import')

Затем нужно запустить ячейки с кодом на выполнение или нажать Ctrl+9. Получится UML-диаграмма как показана на рис. 1. Расположение элементов будет подобрано автоматически.

Пример 2. Даны отношения между классами:

«Дизельный генератор» является подклассом класса «Генератор»

«Газовый генератор» является подклассом класса «Генератор»

«Система электроснабжения» использует «Генератор»

Решение

Аналогично первому примеру можно задать UML-диаграмму через элементы и связи между ними, результат показан на рис. 2.

g = graphviz.Digraph('Classes')

g.node('Генератор', shape='record')
g.node('Дизельный генератор', shape='record')
g.node('Газовый генератор', shape='record')
g.node('Система электроснабжения', shape='record')

g.edge('Газовый генератор', 'Генератор', arrowhead='empty', style='dashed')
g.edge('Дизельный генератор', 'Генератор', arrowhead='empty', style='dashed')
g.edge('Система электроснабжения', 'Генератор', arrowhead='vee', style='dashed', label='<использует>')
