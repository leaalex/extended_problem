Пример 1. Некоторый источник сообщений выдает из алфавита, состоящего из \(8\) символов, информацию. Вероятность появления первого символа \(0.06\), второго – \(0.15\), третьего – \(0.15\), четвертого – \(0.07\) и для пятого, шестого, седьмого, восьмого, соответственно, \(0.05\), \(0.29\), \(0.19\) и \(0.04\). Определить количество информации и ее избыточность.

Решение

Для того, чтобы найти количество информации, воспользуемся формулой Шеннона:

\[ H = - \sum^{N}_{i=1} p_i log_2 p_i,\]

где \(p_i\) - вероятность события.

Найдем количество информации каждого случайного события \(I_i = -log_2 p_i,\) тогда:

\( I_1 = -log_2 0.06 = 4.06, \)

\( I_2 = -log_2 0.15 = 2.74, \)

\( I_3 = -log_2 0.15 = 2.74, \)

\( I_4 = -log_2 0.07 = 3.84, \)

\( I_5 = -log_2 0.05 = 4.32, \)

\( I_6 = -log_2 0.29 = 1.79, \)

\( I_7 = -log_2 0.19 = 2.4, \)

\( I_8 = -log_2 0.04 = 4.64. \)

Вычислим энтропию каждого источника \(H_i = I_i p_i\):

\(H_1 = I_1 p_1 = 4.06 \cdot 0.06 = 0.2436,\)

\(H_2 = I_2 p_2 = 2.74 \cdot 0.15 = 0.411,\)

\(H_3 = I_3 p_3 = 2.74 \cdot 0.15 = 0.411,\)

\(H_4 = I_4 p_4 = 3.84 \cdot 0.07 = 0.2688,\)

\(H_5 = I_5 p_5 = 4.32 \cdot 0.05 = 0.216,\)

\(H_6 = I_6 p_6 = 1.79 \cdot 0.29 = 0.5191,\)

\(H_7 = I_7 p_7 = 2.4 \cdot 0.19 = 0.456,\)

\(H_8 = I_8 p_8 = 4.64 \cdot 0.04 = 0.1856.\)

Тогда для источника символов энтропия равна:

\( H = - \sum^{N}_{i=1} p_i log_{2} p_i = 0.2436 + 0.411 + 0.411 + 0.2688 + 0.216 + 0.5191 + 0.456 + 0.1856 = 2.71 \ бит.\)

Вычислим коэффициент избыточности по формуле:

\[\rho = 1 - \cfrac{H}{log_2 N}\]

где \(H\) - энтропия реального сообщения, \(N\) – количество символов.

Тогда количество информации составляет \(2.71 \ бит\), а ее избыточность \(0.097\).

Пример 2. Для того, чтобы определить хорошие аккумуляторы или нет, необходимо протестировать его работоспособность. Вероятность того, что аккумулятор исправен составляет \(97 \ \%.\) Какое количество аккумуляторов из партии нужно проверить, чтобы ошибка выборки не превышала \(10 \ \%,\) \(5 \ \%\) и \(1 \ \%\) с уровнем доверительности \(99 \ \%\).

Решение

Для того, чтобы найти объем выборки, воспользуемся формулой:

\[ n = \cfrac{\gamma^2 \cdot p \cdot q}{e^2},\]

где \(\gamma\) - нормированное отклонение, зависящее от выбранного уровня доверительного интервала, \(p\) - вероятность события, \(q\) - вероятность обратного события, \(e\) - вероятность ошибки.

По условию задачи уровень доверительности составляет \(99 \ \%\). Найдем значение нормированного отклонения, воспользовавшись таблицей интегральной функции Лапласа. Для \( \Phi \left(\cfrac{p}{2} \right) = \Phi\left(\cfrac{0.99}{2} \right) = \Phi(0.495) \Rightarrow \gamma = 2.58 \).

\( p = 0.97, \ q = 1-p = 0.03. \)

Тогда, если ошибка не превышает \(10 \ \%\), т.е. \(e = 0.1\), получаем:

\[ n = \cfrac{(2.58)^2 \cdot 0.97 \cdot 0.03 }{(0.1)^2} = 19.37 \approx 20 \ (шт.).\]

Тогда, если ошибка не превышает \(5 \ \%\), т.е. \(e = 0.05\), получаем:

\[ n = \cfrac{(2.58)^2 \cdot 0.97 \cdot 0.03 }{(0.05)^2} = 77.48 \approx 78 \ (шт.).\]

Тогда, если ошибка не превышает \(1 \ \%\), т.е. \(e = 0.01\), получаем:

\[ n = \cfrac{(2.58)^2 \cdot 0.97 \cdot 0.03 }{(0.01)^2} = 1937.01 \approx 1938 \ (шт.).\]
