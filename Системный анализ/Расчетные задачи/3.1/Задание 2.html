Какое количество аккумуляторов из партии нужно проверить, чтобы ошибка выборки не превышала \(10 \ \%\), \(5 \ \%\) и \(1 \ \%\) с уровнем доверительности \(95 \ \%\).

Количество аккумуляторов для ошибки \(10 \ \% = \) шт.
Количество аккумуляторов для ошибки \(5 \ \% = \) шт.
Количество аккумуляторов для ошибки \(1 \ \% = \) шт.
