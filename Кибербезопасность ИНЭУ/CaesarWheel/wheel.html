Цель: установить связи между дешифрованным и шифрованным сообщением с помощью кода Цезаря.

Рекомендации по выполнению задания:

  1. Ознакомьтесь с описанием способа кодирования информации шифром Цезаря;
  2. Проанализируйте возможности шифрования и дешифрования сообщений с использованием колец с алфавитом (внутреннее кольцо является исходным алфавитом);
  3. Впишите зашифрованное сообщение для представленного текстового сообщения и дешифрованное для представленного зашифрованного сообщения.

Одним из самых известных и одновременно самых простых способов кодирования информации является шифр Цезаря.

В данном шифре используется циклический сдвиг заданного алфавита на фиксированное значение знаков. В результате циклического сдвига исходного алфавита формируется однозначное соответствие между символами исходного и нового алфавита, при помощи которого шифруется сообщение. Таким образом для корректной расшифровки закодированного сообщения необходимо знать исходный алфавит и ключ кода, которым является значение сдвига.
